ARCHITECTURE.md

智能体构建流程

从概念到部署——四个步骤构建一个生产就绪的 AI Agent

01

🎯 DEFINE — 定义目标

明确智能体的职责边界、输入输出、成功标准。设计 System Prompt 与角色设定，确定所需工具与技能范围。

> claude --system "你是一个代码审查专家..." --skills code-review,simplify
02

🔧 BUILD — 构建技能

编写 Skill 定义文件，配置工具链与 MCP 服务器，设计 Agent Loop 的控制流与错误处理策略。

> cat .claude/skills/my-agent/SKILL.md > claude --skill my-agent "分析这个项目"
03

🧪 TEST — 对抗评测

使用 Judge Panel 多维度评估、对抗验证 (Adversarial Verify)、回归测试套件。Loop-until-dry 模式确保覆盖率。

> claude --skill verify "检查最新变更" --effort max
04

🚀 DEPLOY — 部署迭代

集成到 CI/CD 流水线，配置监控与告警，建立反馈循环。持续收集用户反馈，迭代优化 Agent 表现。

> claude --skill deploy --env production --monitor enabled
